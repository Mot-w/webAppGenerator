### 使用步骤

####    clone

```js
    https://github.com/Mot-w/webAppGenerator.git
```

####    安装依赖

```javascript
    npm i
```

####    初始化项目

```javascript
    gulp init
```

####    开发

```javascript
    gulp dev
```

####    生产

```javascript
    gulp build
```

####    打包

```javascript
    gulp zip
```
